# React Intl SSR Performance Reproduction

## Requisites

- `nvm` installed

## Scenarios

### node 11.15 + react-intl 3.3.2

```zsh
nvm install 11.15
npm start
```

### node 11.15 + react-intl 3.1.13

```zsh
nvm install 11.15
yarn upgrade react-intl@3.1.13
npm start
```

### node 12.10 + react-intl 3.3.2

```zsh
nvm install 12.10
npm start
```
